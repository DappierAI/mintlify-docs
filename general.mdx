#### Platform API Key

To use the Dappier AI Data Models, you will need an API key. Please visit https://platform.dappier.com/ to log in and create an API key in your profile.

<img
  className="block"
  src="/images/dappier_profile_api_key.jpg"
  alt="Dappier Profile"
/>

Our API Key will have the following format: **ak_\*\*\*p51f**. It's a bearer token.

Once you have obtained a bearer token, you need to include it in the header of your HTTP requests to access protected resources. The bearer token is included in the "Authorization" header using the "Bearer" scheme.

#### Using Real Time Data API

<CodeGroup>

```bash Python
import requests
import json

url = "https://api.dappier.com/app/datamodel/dm_01hpsxyfm2fwdt2zet9cg6fdxt"

payload = json.dumps({
  "query": "Hello Dappier!"
})
headers = {
  'Content-Type': 'application/json',
  'Authorization': 'Bearer <YOUR_DAPPIER_API_KEY>'
}

response = requests.request("POST", url, headers=headers, data=payload)

print(response.text)
```

```bash cURL
curl -L 'https://api.dappier.com/app/datamodel/dm_01hpsxyfm2fwdt2zet9cg6fdxt' \
-H 'Content-Type: application/json' \
-H 'Authorization: Bearer <YOUR_DAPPIER_API_KEY>' \
-d '{"query": "Plan me an itinerary for a 3 day trip to Puerto Rico"}'
```
